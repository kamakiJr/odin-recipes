# odin-recipes

This repository contains a collection of simple recipe web pages created as part of The Odin Project curriculum. It is designed to help beginners practice HTML, CSS, and basic web development skills by building and organizing recipes in a structured format.

## Purpose

The goal of this project is to:
- Learn and apply foundational web development concepts
- Practice working with HTML elements, links, images, and lists
- Gain experience with Git and GitHub version control
- Build a basic website to showcase recipes

## Structure

The repository includes:
- An index page listing available recipes
- Individual recipe pages with ingredients and instructions
- Images and supporting assets

## Getting Started

Clone the repository and open the index.html file in your browser to view the recipes. Feel free to add your own recipes or modify existing ones as you learn.
